ci.dfR Documentation

Example dataframe of confidence intervals from bootstrapping.

Description

Example dataframe of confidence intervals from bootstrapping. This file contains a dataframe with 22 columns and 2 rows. The rownames are the Ht x-value for plotting. The columns are the upper and lower confidence intervals.

Usage

ci.df

Format

data.frame

Source

ci.df<-data.frame(do.call(cbind(boot.out[[3]])))
